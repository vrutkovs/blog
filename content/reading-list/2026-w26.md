# Software Engineering & Cloud

* [Life is too short for a slow terminal](https://mijndertstuij.nl/posts/life-is-too-short-for-a-slow-terminal/)

  Skip oh-my-zsh, cache completions with compinit, lazy-load nvm and kubectl, use pure for async git prompt, pick Ghostty. Measuring startup with `time zsh -i -c exit`, hyperfine, and zprof turns vague slowness into fixable bottlenecks.

  Tags: Terminal, Zsh, Performance, Optimization, Shell

* [Running opencode inside openshell on Fedora 44 with llama-cpp](https://blog.tomecek.net/post/opencode-openshell-fedora-llama-cpp/)

  Getting OpenCode inside an OpenShell sandbox on Fedora 44 with local llama.cpp requires fighting rootless Podman, SELinux, port binding, UID mapping, and authentication. All seven blockers documented and solved.

  Tags: OpenShell, Podman, Fedora, Llama, AI
