# Tech Industry & Culture

* [What Is a Dickover?](https://daringfireball.net/2026/05/what_is_a_dickover)

  dickover - a modal panel, popover, or curtain presented by a website or app, deliberately obscuring its own content to frustrate the user with an unwanted, unnecessary, mandatory interaction; e.g. asking the user to accept “cookies”, subscribe to a newsletter, install the website’s mobile app, agree to terms of service, or anything else that the user couldn’t give two shits about.

  Tags: Dickover, Website, User, Design, Modal

* [The circus freaks of open source](https://drewdevault.com/blog/Circus-freaks-of-FOSS/)

  Open source communities are bad at handling public mental health crises. Turning struggling peers into spectacle, harassment targets, or morality plays helps nobody. Compassion, privacy, and restraint beat gawking at flameouts for entertainment.

  Tags: Mental, Source, Harassment, Compassion, Community

* [Who causes the biggest biometric spikes?](https://x.com/the2ndfloorguy/status/2064704204166635930)

  A hobby project connected Whoop stress data to work calendar entries to rank which coworkers caused the biggest biometric spikes. Funny personal hacking, grim product idea: workplace health telemetry turns nasty fast once managers discover dashboards.

  Tags: Whoop, Stress, Analytics, Project, Coworker

# Artificial Intelligence & Society

* [Vague AI Disclosures Aren’t Helping Anyone](https://aftermath.site/crazy-taxi-world-tour-tomb-raider-steam-ai/)

  Steam's AI disclosure boxes create more suspicion than clarity when publishers say almost nothing about what was generated, trained, edited, or replaced. In games, vague compliance reads less like transparency and more like liability management.

  Tags: AI, Game, Disclosure, Controversy, Developer

* [The Jqwik Anti-AI Affair](https://blog.johanneslink.net/2026/06/09/the-jqwik-anti-ai-affair/)

  The `jqwik` maintainer explains why he shipped an anti-AI message aimed at coding agents and how the protest escalated into accusations, legal threats, and project fallout. The mess exposes two problems at once: agentic coding has prompt-injection-shaped failure modes, and open source maintainers are being drafted into unpaid AI supply chains.

  Tags: AI, Ethics, Protest, FOSS, Scandal

* [Michael Tsai - Blog  -  No Siri AI in EU](https://mjtsai.com/blog/2026/06/10/no-siri-ai-in-eu/)

  Apple will skip Siri AI on iOS, iPadOS, and watchOS 27 in the EU, blaming Digital Markets Act requirements. The Commission says Apple made the choice itself. Result: another round of platform politics where privacy, interoperability, regulation, and PR all get folded into one product availability fight.

  Tags: Apple, Europe, Siri, Regulation, AI

* [Why are cached input tokens cheaper with AI services?](https://xeiaso.net/notes/2026/why-llm-cached-token-cheaper/)

  Cached input tokens cost less because providers can reuse key-value cache state for repeated prompt prefixes instead of recomputing the same attention work. Stable system prompts and conversation prefixes turn into real GPU savings, so pricing finally reflects an implementation detail.

  Tags: Cache, AI, Pricing, Token, Inference

* [Do AGENTS.md Files Actually Help Coding Agents?](https://x.com/rasbt/status/2063649136323252397)

  Early research on `AGENTS.md`-style repository context files shows mixed value: developer-written files help more than generated ones, but no context file can be faster and more efficient. Extra instructions make agents do more work, not necessarily better work. Short, specific guidance still matters.

  Tags: Agent, LLM, Research, Evaluation, Context

# Software Engineering & Cloud

* [Stop Using Conventional Commits](https://sumnerevans.com/posts/software-engineering/stop-using-conventional-commits/)

  Conventional Commits optimize for changelog automation while making the least useful part of a commit message mandatory. Scope matters more than type when debugging, reviewing, or handling incidents. Linux- and Git-style scoped subjects carry better operational signal than `feat:` and `fix:` prefixes.

  Tags: Commit, Git, Scope, Standard, Software

* [Helm 3 End of Life](https://helm.sh/blog/helm-v3-end-of-life/)

  Helm 3 has dates now: final feature release in September 2026, security fixes until February 2027. Helm 4 brings WebAssembly plugins, better resource monitoring, multi-document values, server-side apply, and small breaking changes. Time to test charts and CI pipelines before migration becomes urgent.

  Tags: Helm, Migration, Security, Release, Timeline

* [I realized that A cheap VPS is a good front](https://ergaster.org/thoughts/front-server/)

  I host services at home, but I don't want to give me ip to the world. A small VPS and haproxy protect me.

  Tags: Server, VPS, Security, Network, Wireguard
