# Tech Industry & Culture

* [Premium: The Hater's Guide To SoftBank](https://www.wheresyoured.at/premium-the-haters-guide-to-softbank/)

  Tags: SoftBank, Investment, Finance, OpenAI, AI

* [Democratizing Abandonware - Evangelos “GeopJr” Paterakis](https://geopjr.dev/blog/democratizing-abandonware)

  Flathub's LLM-submission ban is mostly reviewer triage, not ideology. Volunteer maintainers keep getting flooded with generated, barely maintained packages; the "AI Slop" tag mostly points at abandoned repos and time nobody gets back.

  Tags: Flathub, AI, Review, Volunteer, Abandon

* [Post by stefano](https://rpi0w.stefanomarinelli.it/@stefano/687a61e9-dae7-4ed2-8c2e-94dfe0bd550c)

  LittleFedi is a new fediverse software demonstrating full functionality on a Raspberry Pi Zero W running NetBSD, using only 33 MB RSS. It provides a comprehensive set of features, including ActivityPub federation, Mastodon API compatibility, streaming, push notifications, and moderation tools, all in a single, lightweight binary. The project highlights its minimal resource footprint while ensuring scalability for diverse hardware.

  Tags: Fediverse, Lightweight, Raspberry, NetBSD, Software

  Tags: AI, Ethics, Work, OpenSource, Dilemma

  Tags: OpenAI, ChatGPT, Browser, Shutdown, AI
