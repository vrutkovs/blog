# Software Engineering & Cloud

* [gittuf - a signed log for git refs](https://nesbitt.io/2026/06/04/gittuf-a-signed-log-for-git-refs.html)

  `gittuf` provides a signed log for Git references, addressing the inherent vulnerabilities of forge-dependent branch protection. Unlike existing systems susceptible to server compromises and admin takeovers, gittuf records every ref update as a cryptographically signed entry within the repository, enabling independent verification of ref movements against a defined policy. This ensures the integrity of the Git tree state, complementing artifact-signing mechanisms by providing a trustworthy source history. The system allows for policy delegations and multi-signature requirements, enhancing software supply chain security outside the forge's direct control.

  Tags: Gittuf, Security, Git, Protection, Policy

* [IPv6 zones in URLs are a mistake](https://xeiaso.net/notes/2026/ipv6-zones-go-url/)

  IPv6 zone IDs in URLs create a nasty standards-compliant trap: the `%` in `%eth0` collides with URL percent-encoding, so it must become `%25eth0`. RFC 6874 blesses the edge case, but framework and browser behavior remains full of sharp edges.

  Tags: IPv6, URL, Encoding, Network, Protocol
