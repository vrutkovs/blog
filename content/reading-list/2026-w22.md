* [Protestware for coding agents](https://nesbitt.io/2026/05/28/protestware-for-coding-agents.html)
  
The `jqwik` 1.10.0 library introduced novel "protestware" by printing an anti-AI message to stdout, visible only to coding agents and CI logs while hidden from human terminals. This open resistance against generative AI creates a new supply-chain challenge for security tooling, which typically overlooks targeted textual output in favor of detecting traditional malicious code execution.

Tags: Protestware, AI, SupplyChain, Security, Software

# Software Engineering & Cloud

* [Jira IS Turing-Complete](https://seriot.ch/computation/jira.html)
  
Atlassian's project-tracking tool Jira is Turing-complete, demonstrated by building a Minsky register machine using its built-in automation features. By mapping machine registers to counts of linked issue types and instructions to automation rules, standard computational conventions allow Jira to function as a fully capable programming language, executing complex sequences like Fibonacci generation.

Tags: Jira, Minsky, Automation, Computation, Register
